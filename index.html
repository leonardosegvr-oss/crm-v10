⚙️ Configurações do Backend

Como conectar ao backend:
1. Cole a URL do seu servidor (Railway, Render, etc.)
2. Cole o token JWT (obtenha rodando node crm-3lg... e fazendo login via Postman/cURL)
3. Clique em Salvar — os dados vão carregar automaticamente.
Para obter o token: faça POST em /api/auth/login e copie o campo token da resposta.
⚠ Lembre-se de configurar CORS no backend antes — adicione logo após const app = express():
app.use((req,res,next)=>{
 res.header('Access-Control-Allow-Origin','*');
 res.header('Access-Control-Allow-Methods','GET,POST,PUT,DELETE,OPTIONS');
 res.header('Access-Control-Allow-Headers','Content-Type,Authorization');
 if(req.method==='OPTIONS') return res.status(200).end();
 next();
});
